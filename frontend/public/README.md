# Public Assets Directory

This directory contains static assets that are served directly by Vite.

## Logo Image

Please place the Cubiq Solutions logo image here as `cubiq-logo.png` (or `.svg`, `.jpg`, etc.)

The receipt will automatically reference this image at `/cubiq-logo.png`.

### Supported Formats:
- PNG (recommended for logos with transparency)
- SVG (scalable, best quality)
- JPG/JPEG

### Recommended Size:
- Height: 48-64px
- Width: Auto (maintain aspect ratio)

If the logo image is not found, the receipt will display without the logo but will still show the website URL and tagline.
